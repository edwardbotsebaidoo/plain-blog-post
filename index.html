Tag Types

p

The paragraph tag. Used to make text with blank space line after.

strong

Boldens text. Keep in mind it is semantic, so information is tied along with this!!

em

Italicizes text. Keep in mind it is semantic, so information is tied along with this!!

h

The heading tag. Has sizes ranging 1-6, with the number being placed after.

meta

Included in the head tag. It tells meta data that goes along with the webpage such as the character set

title

Name of the webpage that defualts in browsers. By default it is the name of the HTML file

html

All code is stored inside these tags!

doctype

References what type of HTML is being used. "!DOCTYPE html" is used to represent HTML 5

head

Where data that isnt displayed like the metadata or the title is stored.

body

Where all content is. All visual elements should be nested inside this.
